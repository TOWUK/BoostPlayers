# BoostPlayers
Плагин для сервера Mindustry (ultra optimized)

Отполирован код (1.41 кб / 10 строк), при входе всем игрокам дает эффект overdrive (бесконечно).
Работает асинхронно, на мобильниках тоже проверено, отлично синхронизируется.

Протестировано на сервере с 1 ядром и 300 мб оперативки (боже, задонатьте мне пж), Потребляет ~1 кб ОЗУ на игрока.

Компилируйте сами, моментально компилируется: `gradle clean jar` или ждите обновления раз в месяц :(
